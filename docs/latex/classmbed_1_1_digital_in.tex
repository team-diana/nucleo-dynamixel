\subsection{Detailed Description}
A digital input, used for reading the state of a pin

Example\+: 
\begin{DoxyCode}
\textcolor{comment}{// Flash an LED while a DigitalIn is true}

\textcolor{preprocessor}{#include "mbed.h"}

\hyperlink{classmbed_1_1_digital_in_a6593a15bcecfd3666c49bba83c81ee9b}{DigitalIn} enable(p5);
DigitalOut led(LED1);

\textcolor{keywordtype}{int} main() \{
    \textcolor{keywordflow}{while}(1) \{
        \textcolor{keywordflow}{if}(enable) \{
            led = !led;
        \}
        wait(0.25);
    \}
\}
\end{DoxyCode}
